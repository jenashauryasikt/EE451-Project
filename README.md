# EE451-Project

### Complie the code
`make`

### Run the serial code
`./serial`

### Run the parallel code
`./async_read_pthreads <NUM_OF_WORKER_BLOCKS>`

### Run using batch files
#### For serial code
`sbatch serial.sl`

#### For parallel code
`sbatch project_1.sl`
`sbatch project_2.sl`
`sbatch project_4.sl`
`sbatch project_8.sl`
